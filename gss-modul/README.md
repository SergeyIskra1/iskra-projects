# ГСС-Модуль Wireframe

Кликабельный черно-белый прототип сайта для производства модульных зданий и блок-контейнеров.

## Команды

```bash
pnpm install
pnpm dev
pnpm build
```

## Деплой

Проект собран на Vite + React и готов к деплою на Vercel:

- Build command: `pnpm build`
- Output directory: `dist`

## Что уже заложено

- многостраничная структура через SPA-роутинг;
- главная страница как вход в каталог и ключевые разделы сайта;
- отдельные URL для каталога, категорий, карточек объектов, услуг, блога, контактов и страницы о компании;
- выпадающие меню каталога и услуг по наведению;
- карточки каталога с переходом в раздел, карточку объекта и расчет;
- услуги как отдельный раздел со страницами услуг;
- модальные формы заявок;
- `vercel.json` с rewrite для прямого открытия внутренних URL.
